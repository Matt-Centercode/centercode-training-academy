# Centercode Training Academy

Interactive, role-based training on the Centercode platform. Learn at your own pace, test out of topics you already know, and earn a verified certificate.

## What It Is

A self-contained web app for onboarding and training Centercode users. Learners create a profile, select a role track, and work through modules at their own pace. Progress is saved locally in the browser.

**At a glance:**
- 10 modules
- 4 role tracks
- 80+ quiz questions
- AI Knowledge Interview

## Viewing It

The live site is hosted on GitHub Pages:

## Running Locally

1. Clone the repo
2. Open a terminal in the project folder
3. Run: `python -m http.server 8080`
4. Open [http://localhost:8080](http://localhost:8080)

## Making Changes

1. Edit files locally
2. Open VS Code → Source Control → stage, commit, push
3. GitHub Pages picks up changes automatically within a few minutes

## Project Structure
